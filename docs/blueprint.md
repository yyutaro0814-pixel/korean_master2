# **App Name**: LexiLink

## Core Features:

- 単語検索と定義: ユーザーが英単語を入力し、その定義、類義語、および発音を表示できるようにします。
- ランダム単語探索: 毎日新しいランダムな英単語とその意味をユーザーに提示し、語彙を発見できるようにします。
- AI生成による例文: 特定の単語に対して、文脈に即した多様な例文を生成するAIツール。
- パーソナライズされた単語リスト: ユーザーは興味深い単語や難しい単語を、素早く復習できるようローカルに保存された個人リストに追加できます。

## Style Guidelines:

- 知的明瞭さと落ち着きを伝えるくすんだティールブルー (#579399)。
- 清潔で開放的なキャンバスを提供する、非常に明るく彩度の低いクールなグレーブルー (#ECF2F3)。
- 主要なアクションやインタラクティブな要素を強調する中間の紫がかった青 (#6699CC)。
- 見出しと本文フォント: 優れた可読性と中立的で客観的な印象を与える、モダンなグロテスクサンセリフ書体「Inter」。
- インターフェースを煩雑にすることなく意味を明確に伝える、クリーンでミニマリストなラインアイコンを使用します。
- 単語と定義に焦点を当てるため、十分な余白を持つミニマリストなカードベースのレイアウト。
- 単語間を移動したり、追加情報を表示したりする際の、さりげなく迅速な遷移により、ユーザーの流れを妨げずに強化します。
